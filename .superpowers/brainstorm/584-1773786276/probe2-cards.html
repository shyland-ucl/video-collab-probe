Design Decisions — Probe 2

  • Creator Mode reuses Probe 1's card pattern exactly — just adds Mode Bar card and Voice Note card on top
  • Helper Mode Bar — Notify + Return buttons now live inside the mode card (not cramped in the header bar)
  • Task Queue stays as its own orange card — it's the helper's primary reference
  • Video Editor card — wraps Video + Transport + Editor toolbar + Timeline in one "Video Editor" card so they feel unified
  • No Marked Segments panel (removed per Probe 1 decision)
