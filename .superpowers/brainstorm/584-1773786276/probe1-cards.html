Design Decisions

  • Primary card (Scene Description) gets the blue (#2B579A) border — it's the most important
  • Secondary cards (Detail Level, Actions) get gray (#64748b) borders — supportive
  • Marked Segments gets amber border — matches existing mark color scheme
  • Each card has a labeled header so screen readers announce the group purpose
  • Nav buttons live inside the description card — they navigate descriptions, so they belong there
  • Video + transport stays as a unified block (already well-grouped)
