How the Text Overlay Works

  • Activate: Tap the "T Text" button in the editor toolbar (highlighted yellow when active)
  • Type: Text Overlay settings card appears below with text input field
  • Style: Pick size (S/M/L) and color from swatches
  • Position: Drag the text element directly on the video frame (dashed border shows it's draggable)
  • Confirm: "Apply Text" locks it in place, "Remove" deletes it
  • Notify Creator removed — Return Device is the only mode action
